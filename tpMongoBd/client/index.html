Book list (avec React)

Le contenu de la page est générée par le client à partir de la réponse JSON reçue depuis http://127.0.0.1:3000/books suite à une requête fetch.

Le code client est géré par React (avec les hooks), voir les composants dans /client/components. Le composant principal est BookApp, c'est ce composant qui exécute la requête fecth à son chargement.
